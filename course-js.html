Khóa học JavaScript căn bản

1. Giới thiệu JavaScript

JavaScript là ngôn ngữ lập trình phổ biến trên web, giúp tạo ra các tương tác động, xử lý sự kiện và xây dựng các ứng dụng web hiện đại.

2. Biến và kiểu dữ liệu

Biến dùng để lưu trữ dữ liệu trong chương trình. JavaScript có ba cách khai báo biến: var, let và const. Các kiểu dữ liệu phổ biến bao gồm số, chuỗi, boolean, object và array.

let name = "Nguyễn Văn A";    // Chuỗi (string)
const age = 25;               // Số (number)
let isStudent = true;         // Boolean (true/false)

3. Toán tử và biểu thức

Toán tử được sử dụng để thực hiện các phép toán trên biến và giá trị. JavaScript hỗ trợ các toán tử số học, so sánh, logic và gán.

let a = 10;
let b = 5;
let sum = a + b;             // 15
let isEqual = (a === b);     // false
let isGreater = (a > b);     // true

4. Câu điều kiện

Câu điều kiện giúp kiểm tra và quyết định luồng xử lý trong chương trình.

let score = 8;
if (score >= 5) {
  console.log("Đậu");
} else {
  console.log("Rớt");
}

5. Vòng lặp

Vòng lặp cho phép thực hiện một đoạn mã nhiều lần. JavaScript hỗ trợ các vòng lặp như for, while và do...while.

// Vòng lặp for
for (let i = 0; i < 5; i++) {
  console.log("Lặp lần thứ " + i);
}

6. Hàm và gọi hàm

Hàm là tập hợp các câu lệnh thực hiện một nhiệm vụ cụ thể và có thể tái sử dụng. Hàm được khai báo bằng từ khóa function.

function greet(name) {
  alert("Chào " + name);
}

greet("Nguyễn Văn A");  // Gọi hàm greet với đối số là tên

7. Làm việc với DOM

DOM (Document Object Model) là cấu trúc của trang HTML. JavaScript có thể truy cập và thay đổi nội dung, kiểu dáng của trang thông qua DOM.

let heading = document.querySelector('h1');
heading.textContent = "Tiêu đề mới bằng JavaScript";

8. Xử lý sự kiện

JavaScript cho phép bạn thêm các sự kiện để phản hồi hành động người dùng, ví dụ như nhấn nút, di chuột, nhập liệu,...

Dưới đây là ví dụ về cách thêm sự kiện nhấn nút:

<button id="btn">Nhấn tôi</button>

<script>
  document.getElementById('btn').onclick = function() {
    alert("Bạn đã nhấn nút");
  }
</script>

Khi người dùng bấm vào nút, sẽ hiện ra thông báo.

9. Tổng kết

Qua khóa học này, bạn đã nắm được những kiến thức cơ bản về JavaScript, bao gồm biến, kiểu dữ liệu, toán tử, câu điều kiện, vòng lặp, hàm, DOM và xử lý sự kiện. Đây là nền tảng vững chắc để tiếp tục học các chủ đề nâng cao hơn.
