Khóa học HTML cơ bản

1. Giới thiệu HTML

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu dùng để xây dựng cấu trúc trang web.

2. Cấu trúc cơ bản của trang HTML

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <title>Tiêu đề trang</title>
</head>
<body>
  <h1>Chào mừng đến với HTML cơ bản</h1>
  <p>Đây là đoạn văn bản.</p>
</body>
</html>

3. Một số thẻ HTML phổ biến

  • <h1> đến <h6> - Tiêu đề với mức độ quan trọng khác nhau
  • <p> - Đoạn văn bản
  • <a href="..."> - Liên kết
  • <img src="..." alt="..."> - Hình ảnh
  • <ul> và <li> - Danh sách không thứ tự

4. Ví dụ đơn giản

<h2>Danh sách mua sắm</h2>
<ul>
  <li>Táo</li>
  <li>Chuối</li>
  <li>Cam</li>
</ul>

5. Tổng kết

Bạn đã làm quen với HTML cơ bản, cấu trúc trang web và một số thẻ hay dùng.
