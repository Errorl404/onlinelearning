Khóa học CSS cho người mới bắt đầu

1. Giới thiệu CSS

CSS (Cascading Style Sheets) dùng để tạo kiểu và bố cục cho trang web HTML.

2. Cách sử dụng CSS

CSS có thể nhúng trong thẻ <style> hoặc file .css riêng.

<style>
  body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
  }
  h1 {
    color: #ffcc00;
  }
</style>

3. Một số thuộc tính CSS cơ bản

  • color: màu chữ
  • background-color: màu nền
  • font-size: kích thước chữ
  • margin, padding: khoảng cách
  • border: viền

4. Ví dụ về bố cục đơn giản

.container {
  width: 80%;
  margin: 0 auto;
  padding: 20px;
}

5. Tổng kết

CSS giúp bạn làm đẹp trang web và tạo bố cục dễ nhìn.
