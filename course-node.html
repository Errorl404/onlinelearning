Khóa học Node.js cơ bản

1. Giới thiệu Node.js

Node.js là môi trường chạy JavaScript trên server, giúp xây dựng ứng dụng backend.

2. Cài đặt Node.js

Tải Node.js từ trang chính thức và cài đặt.

3. Viết server đơn giản

const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Xin chào từ server Node.js!');
});

server.listen(3000, () => {
  console.log('Server chạy tại http://localhost:3000');
});

4. Chạy server

Chạy lệnh node app.js trong terminal.

5. Tổng kết

Bạn đã biết cách tạo server cơ bản bằng Node.js.
