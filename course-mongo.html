Khóa học MongoDB - cơ sở dữ liệu NoSQL

1. Giới thiệu MongoDB

MongoDB là cơ sở dữ liệu NoSQL dùng tài liệu (document) để lưu trữ dữ liệu.

2. Cài đặt MongoDB

Tải MongoDB Community Server và cài đặt.

3. Các thao tác cơ bản

use mydb
db.createCollection("courses")
db.courses.insertOne({title: "MongoDB Basics", description: "Học MongoDB"})
db.courses.find()

4. Sử dụng với Node.js

const { MongoClient } = require('mongodb');
const uri = "mongodb://localhost:27017";
const client = new MongoClient(uri);

async function run() {
  try {
    await client.connect();
    const database = client.db('mydb');
    const courses = database.collection('courses');
    const course = await courses.findOne({});
    console.log(course);
  } finally {
    await client.close();
  }
}
run().catch(console.dir);
